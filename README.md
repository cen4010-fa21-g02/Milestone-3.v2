This vertical prototype addresses the basic functionalties of the website such as the login and registration system which allows to create accounts and login into the website. You may notice that there is not any hobby forums, this is because creating a website such as this requires alot in terms of backend frameworks and code so the seperate hobby forums will be added in the final product. For now, we have integrated teh ability for users to creates forum posts in which they can discuss their hobbies. This is the first step to creating individual forums which will be easier since we have implemented a global forum currently. We have not implemented the store website yet since that requires alot of backend frameworks and API's to implement. We have included a feature that allows users to search for posts based on the tags asscoaited with that post. Later in development, we will add the feature that allows users to search for forums based on the name of the forum. You will also notice that there is not that much css or bootstrap. This is because we mainly focused on the functionality rather than the user interface. Priortizing functionality is essential since users won't enjoy the user interface if the website is not running.



Installation
Type into the terminal:
npm init
npm install
npm i express nodemon mongoose
node app.js

This should run the web server on your local host proxy.
